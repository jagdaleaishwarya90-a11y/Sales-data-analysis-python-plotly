# Sales Data Analysis using Python and Plotly
## Project Overview
This project performs an ene-to-end sales data analysis using python.The goal is to explore sales performance,identify trends and create interactive visualizations using plotly.
## Tools & Technologies
1] Python
2] Pandas
3] Plotly
4] Jupyter notebook
## Dataset
The dataset contains sales information including:
- Data
- Product
- Category
- Price
- Quantity
- Revenue
## Analysis performed
- Data cleaning and preprocessing
- Sales analysis by product and category
- Monthly sales trend analysis
-  Revenue analysis
-  Interactive charts using plotly
## Visualizations
- product-wise sales chart
- category wise revenue chart
- Monthly sales trend
- Top products by revenue

## Key Insights
- Some products generate significantly high revenue than others.
- Sales show trend over different months.
-  certaincategories dominate total sales.

## Conclusion
This project demonstrates how python can be used for data analysis and interactive visualization.plotly helps create dynamic charts that make data insights easier to understand.
